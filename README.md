# @wanglihua/lulu

[Docs](https://qluwlh.github.io/lulu/)

## Installation

To use with node:
```shell
npm i @wanglihua/lulu
```
Then in the console:

```typescript
import { add } from('@wanglihua/lulu');
```

**Note for ES6 module and browsers**
```typescript
import { add } from('@wanglihua/lulu/es6');
```
